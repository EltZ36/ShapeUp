Shape Up is an engaging mobile puzzle game that tests your problem-solving skills through a series of increasingly challenging levels. Players use touch gestures to manipulate shapes and solve unique puzzles in a minimalist, visually appealing environment.

## Features

- **Intuitive Touch Controls**: Simple swipe, tap, and drag mechanics that feel natural on mobile devices
- **Progressive Difficulty**: Gradually increasing challenge across multiple puzzle types
- **Multiple Puzzle Types**: Various puzzle mechanics that keep the gameplay fresh and engaging
- **Clean, Minimalist Design**: Focus on gameplay with a distraction-free interface

## Installation

1. As of right now, clone repo and build for mobile
2. Install the game on your mobile device
3. No additional setup required - start playing immediately!

## How to Play üß©üß©

1. Launch the game
2. Select a level from the available puzzles
3. Use touch controls to manipulate shapes:
   - Swipe to move shapes
   - Pinch to resize
   - Utilize gyroscrope / accelerometer
4. Complete the puzzle objectives to progress
